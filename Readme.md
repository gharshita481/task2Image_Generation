PRODIGY_GA_02

 Task 02 - Image Generation with Pre-trained Models

Objective

The objective of this task is to generate images from text prompts using a pre-trained generative AI model. Stable Diffusion was used to transform natural language descriptions into realistic AI-generated images.

 Technologies Used

* Python
* Google Colab
* Stable Diffusion
* Hugging Face Diffusers
* PyTorch

Steps Performed

1. Installed the required libraries.
2. Loaded the Stable Diffusion pre-trained model.
3. Provided text prompts.
4. Generated images from the prompts.
5. Saved and downloaded the generated images.

Sample Prompts

* A futuristic city at night with flying cars
* A robot studying in a library
* An astronaut riding a horse on Mars
* A beautiful sunset over mountains

Outcome

Successfully generated AI images from text descriptions using the Stable Diffusion model.
